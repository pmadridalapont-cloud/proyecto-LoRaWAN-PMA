# LilyGO T3-S3 LoRaWAN

LoRaWAN example code and configuration for the **LILYGO T3-S3 ESP32-S3 V1.2** development board with the **Semtech SX1276** radio, using the **MCCI LMIC** library and **The Things Network (TTN)**.

This repository provides working Arduino examples for connecting the LilyGO T3-S3 to TTN using OTAA activation.

Detailed setup, configuration, and troubleshooting instructions are available here:

👉 [https://pmanzoni.notion.site/lilygot3s3-lorawan](https://pmanzoni.notion.site/lilygot3s3-lorawan)

## Repository Structure

* `lilygot3s3_ttn_base` — stable and tested version
* `lilygot3s3_ttn_base_v2` — newer development version with ongoing improvements and better documentation in the code
